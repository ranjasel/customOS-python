# CustomOS (Python)

A simple CLI-based file management system built in Python.

## Features

* Create files
* Write content (append or rewrite)
* View file contents
* Delete files
* JSON-based file persistence

## How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run:

   ```
   python customOS.py
   ```

## Notes

* Files are stored using a JSON file for persistence
* This project is a basic simulation of a file system
